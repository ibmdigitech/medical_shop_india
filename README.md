# Amster Med Care - Online Pharmacy Kerala

A modern e-commerce pharmacy website built with React + Vite, featuring free home delivery, special offers, and comprehensive healthcare product listings.

## Features

- **Free Home Delivery** across Kerala with same-day delivery options
- **SEO Optimized** - Structured data for pharmacy/local business
- **Responsive Design** - Works on mobile, tablet, and desktop
- **Dark Mode Support** - Automatic switching based on system preference
- **Product Categories** - Medicines, wellness, lab tests, Ayurvedic, baby care
- **Special Offers** - FLAT 25% OFF, free delivery, same-day delivery
- **Prescription Upload** - Easy prescription submission
- **Medicine Comparison** - Compare up to 4 medicines side-by-side
- **Store Locator** - Find nearby Amster Med Care locations
- **Contact Options** - Phone, WhatsApp, email, contact form

## Technology Stack

- **Frontend:** React 19, Vite 8
- **Styling:** Tailwind CSS 4 with custom medical color theme
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **State Management:** React Context (Cart)
- **Routing:** React Router DOM v7
- **Forms:** Custom form components with validation
- **Carousel:** Swiper.js for testimonials

## Medical Color Theme

- **Primary:** Medical Cyan `#0891B2` (Trust & Healthcare)
- **Secondary:** Medical Green `#059669` (Health & Wellness)
- **Accent:** Urgent Red `#DC2626` (Offers & Alerts)
- **Background:** Clean white/off-white with dark mode support

## SEO & AEO Features

- Schema.org structured data (Pharmacy & LocalBusiness)
- Location-specific meta tags (Omassery, Kerala)
- Offer-focused descriptions with keywords
- Proper heading hierarchy and semantic HTML
- Open Graph and Twitter Card tags
- Canonical URLs and mobile optimization

## Key Pages

- **Home:** Hero with delivery offers, special promotions, trust indicators
- **Medicines:** Product catalog with filtering, search, and discount badges
- **Contact:** Location info, WhatsApp calling, contact form
- **About:** Company story, team, mission
- **Blog:** Health tips, medicine guides, wellness articles
- **Services:** Lab tests, wellness products, healthcare services
- **Cart:** Secure checkout with applied discounts

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=your_api_endpoint_here
VITE_MAPS_API_KEY=your_google_maps_key
```
